oVirt Node

oVirt Node is a minimal operating system based on CentOS that is designed to provide a simple method for setting up a physical machine to act as a hypervisor in an oVirt environment. The minimal operating system contains only the packages required for the machine to act as a hypervisor, and features a Cockpit user interface for monitoring the host and performing administrative tasks.

Installing oVirt Node

Before you start installing oVirt Node be sure to meet requirements on your virtualization host, see Host Requirements in the Installing oVirt as a self-hosted engine using the command line guide.

Once ready, see oVirt Nodes in the Installing oVirt as a self-hosted engine using the command line guide for installation instructions.
