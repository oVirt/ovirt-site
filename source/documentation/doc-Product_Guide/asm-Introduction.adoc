= Introduction to {virt-product-fullname}

:context: RHV_intro

{virt-product-fullname} is an enterprise-grade virtualization platform built on {enterprise-linux}. Virtualization allows users to easily provision new virtual servers and workstations, and provides more efficient use of physical server resources. With {virt-product-fullname}, you can manage your entire virtual infrastructure - including hosts, virtual machines, networks, storage, and users - from a centralized graphical user interface or REST API.
